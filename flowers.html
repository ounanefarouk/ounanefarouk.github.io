Ray-Marched Procedural Flowers

Interactive Procedural Flowers

This interactive visualization features a 3D Flower Proceduraly generated through a Signed distance Function then displayed using ray-marching within the fragment shader.

Interactions:

  • Drag to rotate the Flower
  • Use sliders to interpolate between the three flowers : Camelia, Lotus and Dahlia.
